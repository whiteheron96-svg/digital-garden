# SYNCROOM + Scarlett Solo (2세대) 연결 가이드 — Mac + MainStage

  

> Mac에서 Scarlett Solo 2세대 오디오 인터페이스로 Yamaha SYNCROOM 합주하기.

> 주 용도: **MainStage(소프트음원) + 마이크(말/노래)**, 건반은 USB-MIDI 연결.

  

---

  

## 0. 핵심 요약

  

- **루프백 기능은 필요 없다.** 단, MainStage 같은 *소프트웨어 음원* 소리를 SYNCROOM에 보내려면 **가상 오디오(BlackHole)** 로 라우팅이 필요하다.

- **하드웨어 신디사이저**만 쓸 거라면 BlackHole 없이 케이블로 Scarlett 입력에 바로 꽂으면 된다.

- **헤드폰 필수** (스피커 사용 시 마이크 하울링 발생).

- 인터넷은 **유선 LAN 권장**.

  

---

  

## 1. 준비물 / 드라이버

  

- **Mac**: 별도 드라이버 불필요 (Core Audio). Focusrite Control은 설치 권장.

- (참고) Windows라면 Focusrite USB ASIO 드라이버 필수 — 이 가이드는 Mac 기준.

  

---

  

## 2. BlackHole 설치 (가상 오디오)

  

### Homebrew로 설치 (사용한 방법)

```bash

brew install blackhole-2ch

```

  

### 설치 후 인식

- 전체 재부팅 불필요.

- **'오디오 MIDI 설정'** 앱에서 `BlackHole 2ch`가 보이면 OK.

- 안 보이면 오디오 데몬만 재시작:

```bash

sudo killall coreaudiod

```

⚠️ 실행 순간 시스템 소리가 1~2초 끊김 (다른 작업엔 지장 없음).

  

---

  

## 3. 다중 출력 장치(Multi-Output Device) 만들기

  

**'오디오 MIDI 설정'** 앱에서:

  

1. 좌측 하단 `+` → **'다중 출력 장치 생성'**

2. 체크:

- ✅ Scarlett Solo USB

- ✅ BlackHole 2ch

3. **마스터 장치** = `Scarlett Solo`

4. **드리프트 보정** = `BlackHole 2ch` 쪽 체크

5. 이름 변경(선택): 예) `Scarlett+BlackHole`

  

> Scarlett / BlackHole 둘 다 샘플레이트 **48000 Hz**로 통일.

  

---

  

## 4. MainStage 설정

  

1. MainStage → 환경설정(Preferences) → 오디오(Audio)

2. **출력 장치** = `Scarlett+BlackHole` (다중 출력 장치)

3. **입력 장치** = `Scarlett Solo` (마이크용)

4. **버퍼 사이즈** = 64~128 샘플

  

### 마이크를 MainStage로 받기 (말/노래용)

- Audio 채널 스트립 추가

- 채널 **Input = Scarlett 입력 1번** (XLR 마이크)

- **Input Monitoring 켜기** (내 목소리 모니터)

- (선택) 노이즈 게이트/컴프레서로 말소리 정리

  

---

  

## 5. SYNCROOM 오디오 설정

  

| 항목 | 값 |

|------|-----|

| 입력(Input) | **BlackHole 2ch** |

| 출력(Output) | **Scarlett Solo** |

| 입력 모니터링 | **OFF** |

| 샘플레이트 | **48000 Hz** |

  

> 입력 모니터링 OFF 이유: 내 소리는 이미 다중 출력으로 Scarlett(저지연)에서 직접 듣기 때문. 켜면 겹치고 지연 발생.

  

---

  

## 6. 연결 구조도 (MainStage + 마이크)

  

```

[마이크] ┌──────────────────────────────────────────────┐

│ XLR │ Mac (내부) │

▼ │ │

┌─────────┐ USB │ ┌────────────────────────────┐ │

│Scarlett │───────────────────▶│ MainStage │ │

│ Solo │ IN1=마이크 │ │ ┌──────────┐ ┌──────────┐ │ │

│ │◀──────────────────│ │소프트음원 │ │마이크 채널│ │ │

└────┬────┘ 헤드폰 출력 │ │ │(건반연주) │ │(IN1 받기) │ │ │

│ │ │ └────┬─────┘ └────┬─────┘ │ │

🎧 내 귀 │ │ └─────┬──────┘ │ │

(음원+마이크 │ │ ▼ 합쳐진 출력 │ │

+상대소리) │ │ = 다중 출력 장치 │ │

▲ │ └──────┬──────────────┬──────┘ │

│ │ ▼ ▼ │

│ │ ┌──────────┐ ┌───────────┐ │

│ │ │ Scarlett │ │ BlackHole │ │

│ (상대 소리) │ │(직접모니터)│ │ 2ch │ │

│ │ └────┬─────┘ └─────┬─────┘ │

│ │ │ ▼ │

│ │ │ ┌───────────┐ │

└──────────────────────────────┴─상대◀───┤ SYNCROOM ├──▶ 상대에게 │

│ │In:BlackHole│ 음원+마이크 │

│ │Out:Scarlett│ │

│ └───────────┘ │

└──────────────────────────────────────────────┘

```

  

### 소리 흐름

```

내 소리(음원+말): MainStage → 다중출력 → Scarlett 🎧 (저지연 직접 모니터)

↘ BlackHole → SYNCROOM → 상대

상대 소리: SYNCROOM → Scarlett 🎧

```

  

---

  

## 7. (참고) 하드웨어 신디사이저만 쓸 경우

  

BlackHole 불필요. 신디 Audio Out → Scarlett 입력 단자에 케이블 연결.

  

```

[하드웨어 신디] --오디오 케이블--> Scarlett IN → SYNCROOM(In/Out 모두 Scarlett) → 상대

↘ DIRECT MONITOR로 내 소리 모니터

```

  

- SYNCROOM 입력 = Scarlett, 출력 = Scarlett

- Scarlett **DIRECT MONITOR**로 직접 모니터링

  

---

  

## 8. 동작 테스트 체크리스트

  

1. [ ] MainStage 건반 → 내 헤드폰(Scarlett)에서 소리 들림 → 다중 출력 OK

2. [ ] SYNCROOM 입력 레벨 미터 움직임 → BlackHole 전송 OK

3. [ ] 상대 소리가 헤드폰으로 들림 → SYNCROOM 출력 OK

4. [ ] 마이크 말소리가 미터에 잡힘 → 마이크 채널 OK

  

---

  

## 9. 자주 막히는 부분 (Troubleshooting)

  

- **소리 안 남/깨짐** → MainStage·SYNCROOM·다중출력 내 장치들 샘플레이트를 **모두 48000Hz**로 통일.

- **하울링/에코** → 반드시 헤드폰 사용 (스피커 금지).

- **BlackHole 안 보임** → `sudo killall coreaudiod` 또는 재로그인.

- **내 소리 두 번 들림/지연** → SYNCROOM 입력 모니터링 OFF 확인.

- **건반 인식 안 됨** → USB-MIDI 연결 확인 (5핀 MIDI 단자만 있으면 MIDI 인터페이스 필요).
