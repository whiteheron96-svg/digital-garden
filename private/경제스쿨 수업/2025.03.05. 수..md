## 과제 : 마이클 포터의 5가지 힘 

- AB맵핑에서 5 Forces로 확장되야 함. 
- 5가지 힘으로 쪼개야 한다. 
- 내부(공통재) / 수요 / 공급 / 경쟁자 / 진입장벽
- 공통재가 가운데 있도록 한다! 
- ex. 내부 (시간) / 대체재 : 닌텐도 / 기존 : 나이키 
- 경쟁자의 원리 : 나이키, 아디다스, 퓨마 


### 호가창 보기
- 1. 매도세가 강하고, 매수세가 약할 때 -> 가격이 올라간다 
- 2. 매수세가 강하고, 매도세가 약하다 (그래 내가 파는 사람이 내려올게)
- 3. 역전할 때 : 골든크로스가 일어나는 시점. 
- 가만히 기다리고 있다가 12000원보다 14000원에 던질 때 -> 올라가는 시기. 
- 대량주문의 수가 나온다 : 10000주, 2만주가 나오는지 주시 -> 매도세가 세진다. 
- 거래량이 터지는 아침 시간 (9시 ~ 9시 반) : 5~15%까지 먹을 수 있음. 
- 같은 주식을 1분과 10틱으로 본다. 


- 양봉으로 끝난 주식은 다음날에도 이어질 가능성이 높다.
