
### 오피스텔
오피스텔을 매매하는 사람들은 매매차익이 아니라 '임대수익률'을 목표로 한다. 

### 자료 레퍼런스
- 건폐율 및 용적률 : 자치법규정보 시스템 

### 경매 용어
- 강제경매와 임의경매 : 강제경매는 판결문이나 공정증서(집행권원)을 가지고 경매 신청, 임의경매는 근저당권이나 전세권(담보권)으로 경매 신청 
- 청구금액 : 판결 금액(판결문) 또는 근저당권 채권 금액(근저당권)
- 배당요구종기일 : 배당을 받은 채권자들(임차인 등)이 배당요구를 해야만 하는 마지막 날
	- 배당요구기준일은 ‘경매개시결정기입등기(경매등기!)’ : 이전에 근저당권을 잡은 사람은 배당요구를 하지 않아도 순위에 따라 배당을 받게 되고, 경매등기일 이후는 배당요구를 해야만 배당을 받을 수 있다! 
- 최저매각가격 : 매각당일에 부동산을 매각할 수 있는 최저 가격. 1회 유찰 될 때마다 20~30%씩 떨어진다. 
	- 입찰보증금은 입찰가에 상관없이 최저매각가격의 10%! 
- 현황조사 : 점유자, 점유 이유, 보증금 및 월세 등 조사 
- 차순위매수신고 : 낙찰자가 잔금을 납부하지 않았을 때 다음 사람을 바로 지정
	- 단, 낙찰가액 - 입찰보증금 + 1원 이상 입찰한 사람만 할 수 있음. 
- 매각 조건
- 낙찰률과 낙찰가율 : 부동산 경기 전망의 지표가 된다. 

|        |              |                               |
| ------ | ------------ | ----------------------------- |
| 법정매각조건 | 최저매각가격 미만 불허 | 입찰 시 최저 매각가격 이상으로 응찰해야 함      |
|        | 소멸주의         |                               |
|        | 인수주의         | 낙찰자가 낙찰받고 부동산과 관련된 하자를 인수하는 것 |
|        | 잉여주의         | 경매 신청 채권자가 배당을 받을 때 경매가 유효한 것 |
| 특별매각조건 | 농지취득자격증명원    |                               |
|        | 재매각 시 보증금 할증 | 최초 매각시 10%에서 재매각시 20%로 할증.    |

> 배당요구를 하지 않으면 쫓겨난다. 
> 필요서류인 임대차 계약서와 주민등록초본을 첨부해서 해당 경매계에 제출하자

### 입찰절차
- 준비물 : 신분증, 입찰보증금(수표가 좋음)

> [!NOTE] Title
> 입찰보증금은 1원이라도 부족하면 안된다. 

- 아침에 법원에 가서 하는 일 : 입찰표 작성, 입찰보증금 봉투 및 입찰 봉투 작성 (30분 전에 도착하면 좋다)
 - 매각물건명세서 확인 
- 입찰표 작성 : 사건번호 및 물건번호 기재
